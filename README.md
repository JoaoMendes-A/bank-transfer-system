# Sistema de Conta Bancária em C#

---

## Descrição do Projeto

Aplicação de console desenvolvida em C# com o objetivo de simular um sistema bancário básico.
O projeto permite a criação e o gerenciamento de contas bancárias, aplicando conceitos de
Programação Orientada a Objetos, organização de código e boas práticas de desenvolvimento.

---

## Funcionalidades

- Criação de contas bancárias
- Geração automática de identificador (ID)
- Depósito de valores
- Saque com validação de saldo
- Transferência entre contas
- Consulta de contas (nome do titular, ID e saldo)
- Validação de entradas do usuário
- Código organizado, comentado e padronizado

---

## Tecnologias Utilizadas

- C#
- .NET (Console Application)
- Programação Orientada a Objetos
- Coleções (`List<T>`)
